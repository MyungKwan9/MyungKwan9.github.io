About

현업 운영 경험을 시각화 가능한 구조로 정리합니다.

MK CS/VAN Operations · KPI Visualization

CS/VAN 운영 업무를 실제로 경험하며 현장에서 발생하는 문의, 처리 흐름, 정산 이슈, 운영 비용, 사업장별 문제를 시각화 가능한 업무 구조로 정리하고 있습니다. 개발자는 아니지만 운영 기획을 개선하고 설계하는 방법을 연구 중입니다.

현재는 뉴미디어디자인 전공 학습과 AI 도구 활용을 바탕으로 업무 데이터를 KPI, 화면 구조, 문서화, 대시보드 MVP 형태로 정리하는 역량을 강화하고 있습니다.

Capabilities

운영, 데이터, 화면 설계를 연결하는 핵심 역량

CX에만 한정하지 않고 서비스 운영, CS 운영관리, VOC 분석, KPI 기획, UI/UX 기반 개선까지 연결합니다.

01

CS 운영 이해

고객 문의, 원격지원, 반복 이슈, 처리 흐름을 현업 관점에서 파악하고 정리합니다.

02

VAN / 결제 업무 흐름

거래, 정산, 보류정산, 운영비용, 사업장 이슈 등 결제 운영 데이터를 업무 맥락으로 이해합니다.

03

VOC / 문의 데이터 정리

반복 문의와 운영 이슈를 분류하고 개선 포인트를 찾기 위한 기초 데이터로 정리합니다.

04

KPI 기획

실무 데이터를 거래 횟수, 거래 금액, 보류정산, 운영비용, 사업장별 이슈 같은 판단 지표로 구조화합니다.

05

Dashboard MVP 제작

React, Vite, Recharts를 활용해 운영 데이터를 시각화하는 대시보드 MVP 제작 경험을 쌓고 있습니다.

06

AI 활용 업무 개선

리서치, 문서화, 화면 설계, 코드 제작 보조에 AI를 활용해 실무형 결과물을 빠르게 구성합니다.

Process

문제를 화면으로 바꾸는 작업 방식

단순히 예쁜 화면을 만드는 것이 아니라, 현업 문제를 지표와 화면 구조로 설명할 수 있는 형태로 정리합니다.

01

현업 문제 파악

상담, 운영, 정산, 사업장 이슈에서 반복되는 문제를 찾습니다.

02

데이터 항목 정리

업무 흐름에 필요한 항목과 불필요한 항목을 구분합니다.

03

KPI 기준 설계

거래, 비용, 보류, 이슈 같은 판단 기준을 지표로 구조화합니다.

04

화면 구조화

사용자가 빠르게 이해할 수 있도록 카드, 차트, 요약 영역을 설계합니다.

05

MVP 제작

HTML/CSS 또는 React 기반으로 확인 가능한 결과물을 만듭니다.

06

개선점 정리

완성된 화면을 바탕으로 다음 개선 방향과 확장 가능성을 정리합니다.
