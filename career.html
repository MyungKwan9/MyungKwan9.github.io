커리어 요약

경력의 중심은 단순 상담이 아니라 문제 흐름을 이해하는 경험입니다.

저는 다양한 CS 및 원격지원 환경에서 사용자의 문의, 오류, 장애, 설정 문제를 직접 대응하며 반복되는 문제의 흐름을 경험했습니다. 단순히 문의를 처리하는 것을 넘어, 어떤 문제가 왜 반복되는지 정리하는 관점을 키워왔습니다.

현재는 VAN 결제 운영과 학습 시스템 지원 업무를 바탕으로 결제, 정산, 운영 문의, 사용자 환경 이슈를 이해하고 있으며, 이를 FAQ, KPI, 문서화, 대시보드 MVP 형태로 정리하는 역량을 확장하고 있습니다. 또한 SQLD 준비를 통해 VOC 유형, 운영 이슈, KPI 지표가 어떤 데이터 구조로 정리될 수 있는지 이해하는 데이터 문해력을 강화하고 있습니다.

핵심 경험

경력기술 핵심 경험

공개 포트폴리오에서는 세부 회사 내부 정보와 민감한 업무 데이터는 제외하고, 직무 경험과 문제 해결 방식 중심으로 정리했습니다.

01

CS 운영 및 원격지원

고객 문의 대응, 프로그램 설치 지원, 오류 확인, 원격 환경 점검, 반복 이슈 정리 등 다양한 사용자 지원 경험을 보유하고 있습니다.

02

VAN 결제 운영 이해

결제 데이터 확인, 정산 관련 문의, 보류정산, 사업장 이슈, 대리점 커뮤니케이션 등 VAN 운영 흐름을 실무 관점에서 이해합니다.

03

반복 문의 유형화

반복되는 문의와 장애 유형을 단순 기록으로 남기지 않고, 원인과 처리 흐름을 분류해 개선 가능한 형태로 정리합니다.

04

FAQ / 지식 데이터 구조화

문의 데이터를 질문과 답변 구조로 정리하고, 운영자와 개발자가 활용할 수 있는 형태의 FAQ 및 지식 데이터로 구성하는 경험이 있습니다.

05

KPI 관점의 업무 정리

거래, 정산, 비용, 이슈, 문의 유형처럼 운영 판단에 필요한 항목을 지표 기준으로 정리하고 화면 구조로 연결하는 방향을 학습하고 있습니다.

06

문서화와 협업 전달

상담 현장의 언어를 매뉴얼, FAQ, 요구사항, 화면 구성안처럼 협업 가능한 문서 형태로 바꾸는 역량을 확장하고 있습니다.

대표 경력 흐름

현재 커리어 방향과 연결되는 경험

전체 이력의 모든 회사를 나열하기보다, 현재 커리어 방향과 연결되는 경험을 중심으로 압축했습니다.

현재

CS / VAN 운영 지원

VAN 결제 운영, 정산 관련 문의, 대리점 커뮤니케이션, 학습 시스템 원격지원 및 반복 오류 대응을 수행하고 있습니다.

헬프데스크

기업 사용자 원격지원

기업 내부 사용자 대상 그룹웨어, MS 제품, 사용자 환경 설정, 시스템 오류 관련 상담 및 원격지원 경험을 쌓았습니다.

결제 지원

POS / 결제 장비 지원

편의점 POS 기기와 결제 시스템 오류를 원격으로 점검하고, 운영 환경에서 발생하는 장애 대응 흐름을 경험했습니다.

절차 지원

인증서 / 절차 기반 서비스 지원

인증서 신청, 발급, 저장, 복사 등 절차 기반 서비스에서 발생하는 사용자 혼란과 오류 패턴을 대응했습니다.

제품 CS

제품 상담 / A/S 지원

제품 사용 문의, 초기 설정 안내, 불만 접수, A/S 접수, 신입 상담원 교육 지원 경험을 보유하고 있습니다.

현장 운영

생산 / 품질 기반 경험

생산, 조립, 품질 기준, 장비 구조 이해 경험을 통해 현장 운영과 문제 해결의 기본 감각을 쌓았습니다.

역량

핵심 역량

현업 경험을 바탕으로 요구사항을 정리하고, 화면 구조와 MVP로 검증할 수 있는 실무형 운영·기획 역량에 초점을 둡니다.

01

운영 흐름 이해

CS 상담, 원격지원, 결제 운영, 정산 문의, 사업장 이슈 흐름을 실무 관점에서 이해합니다.

02

이슈 유형 분류

반복 문의, 오류 유형, 사용자 혼란 지점을 분류하고 개선 가능한 기준으로 정리합니다.

03

FAQ / 문서화

질문-응답 구조, 매뉴얼, 업무 안내 문서, 지식 데이터를 재사용 가능한 형태로 정리합니다.

04

KPI 구조화

운영 데이터를 거래, 비용, 정산, 이슈 기준으로 구조화하고, SQLD 준비를 통해 지표와 데이터 항목의 관계를 이해하는 역량을 강화하고 있습니다.

05

대시보드 기획

카드, 차트, 표, 필터 등 사용자가 빠르게 이해할 수 있는 화면 구조를 설계합니다.

06

AI 보조 업무 흐름

리서치, 문서화, 화면 설계, 코드 제작 보조에 AI 도구를 활용해 결과물을 빠르게 실험하고 정리합니다.

업무 활용 도구

문서화, 시각화, MVP 제작 도구

현업 문제를 정리하고 문서화하며, 화면 구조와 포트폴리오 결과물로 연결하기 위해 활용하는 도구를 정리했습니다.

문서화

Notion / Office

FAQ, 업무 흐름, 운영 문서, 회의 자료, 정리 문서 작성에 활용합니다. Excel과 PowerPoint는 데이터 정리와 보고 자료 구성 중심으로 사용합니다.

AI 리서치

GPT / Perplexity

리서치, 문서 초안, 요구사항 정리, 화면 구성 아이디어, 코드 제작 보조와 학습 정리에 활용합니다.

디자인 보조

Figma / Canva / Adobe

Figma는 UI 와이어프레임과 화면 구조 설계 학습 및 포트폴리오 개선에 활용할 예정입니다. Canva는 포트폴리오 시각 자료와 간단한 디자인 구성에 활용하며, Illustrator와 Photoshop은 기초 편집 및 이미지 보정 보조 수준으로 활용하고 있습니다.

웹 / 정적 페이지

HTML / CSS / GitHub Pages

HTML/CSS 기반 정적 페이지 제작, GitHub Pages 배포, Vercel을 활용한 대시보드 MVP 배포 경험이 있습니다.

대시보드 MVP

React / Vite / Recharts

전문 개발자 수준이 아니라, 운영 데이터를 카드, 차트, 표 구조로 시각화하는 MVP 제작 경험 중심으로 활용합니다.

데이터 문해력

SQLD 준비

개발자 전환 목적이 아니라, VOC 유형, 운영 이슈, KPI 지표가 어떤 데이터 구조로 정리될 수 있는지 이해하기 위해 SQLD를 준비하고 있습니다.

강화 방향

뉴미디어디자인 기반 시각화/데이터 역량 확장

한양사이버대학교 뉴미디어디자인 전공 학습을 통해 웹디자인, UI/UX, 데이터 시각화, AI 활용 역량을 실무 경험과 연결하고 있습니다.

현재는 HTML/CSS, UI/UX, 데이터 시각화, React/Vite 기반 MVP 제작, AI 문서화 역량을 실무 경험과 연결해 강화하고 있습니다.

또한 CS/VAN 운영 지원 경험을 CX 기획과 서비스 운영개선 역량으로 확장하기 위해 SQLD를 준비하며, 데이터 구조, 데이터 모델링, SELECT, JOIN, GROUP BY 등 기본 SQL 개념을 학습하고 있습니다.

목적은 개발자 전환이 아니라, VOC 유형, 운영 이슈, KPI 지표가 어떤 데이터 구조로 정리되고 해석될 수 있는지 이해하는 것입니다. 현재 VAN KPI Dashboard MVP와 연결해 거래 성공률, 실패율, 취소율, 정산 이슈, VOC 유형 등 운영 지표를 구조화하는 데이터 문해력을 강화하고 있습니다.

목표는 현업에서 발생하는 업무 흐름과 개선 아이디어를
데이터 구조와 화면 구조로 함께 정리하는 역량을 확장하는 것입니다.
